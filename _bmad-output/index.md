## Project Overview

Multi-part UI + API system for browsing agent personas and running chat workflows. UI integrates with backend via HTTPS; endpoint alignment needed for `/agents` and `/chat`.

## Project Structure

This project consists of 2 parts:

### Web UI (ui)

- **Type:** web
- **Location:** `AgentChain_UI/AgentChainUI`
- **Tech Stack:** React 19 + Vite (TypeScript)
- **Entry Point:** `src/main.tsx`

### Backend API (api)

- **Type:** backend
- **Location:** `Agent_Chain_API`
- **Tech Stack:** ASP.NET Core (.NET 8)
- **Entry Point:** `againt_chain_api/Program.cs`

## Cross-Part Integration

- UI calls API via `VITE_API_BASE_URL`.
- UI expects `GET /agents`, `GET /agents/{id}`, `POST /chat`.
- API exposes `GET /api/agents` and `GET /WeatherForecast`.

## Getting Started

### Web UI Setup

**Prerequisites:** Node.js + npm

**Install & Run:**

```bash
cd AgentChain_UI/AgentChainUI
npm install
npm run dev
```

### Backend API Setup

**Prerequisites:** .NET SDK 8.0

**Install & Run:**

```bash
cd Agent_Chain_API
dotnet build
dotnet run --project againt_chain_api/againt_chain_api.csproj
```
