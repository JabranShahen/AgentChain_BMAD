# Development Guide (UI)

## Prerequisites
- Node.js (LTS recommended)
- npm

## Setup
```bash
cd AgentChain_UI/AgentChainUI
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Lint
```bash
npm run lint
```

## Notes
- Uses Vite (rolldown-vite) with React and TypeScript.
