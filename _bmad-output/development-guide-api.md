# Development Guide (API)

## Prerequisites
- .NET SDK 8.0

## Setup
```bash
cd Agent_Chain_API
```

## Build
```bash
dotnet build
```

## Run
```bash
dotnet run --project againt_chain_api/againt_chain_api.csproj
```

## Tests
```bash
dotnet test
```

## Notes
- Solution: `ai_agent.sln`
- Projects: `ai_agent`, `agent_service`, `againt_chain_api`, `agent_service.Tests`
